# **App Name**: CorePayLinks

## Core Features:

- Currency Selection: Accept payments in AVAX or BTC. The user selects the currency.
- Amount Input: A field to specify the amount to be paid.
- Payment Link Generation: Generate a QR code and deepl-link for Core Wallet, based on payment details.
- QR Code Display: Display the payment link as a QR code.
- Copy to Clipboard: Copy the payment link to the clipboard.
- USDC Estimation: Generates estimated USDC value using live price feed data as a tool.
- Payment Status Indication: Visually show the payment is pending, and switch the indicator to 'confirmed' when the simulated payment detection is triggered.

## Style Guidelines:

- Primary color: HSL(150, 50%, 50%) which translates to approximately #40BF80 for a techy, modern look.
- Background color: HSL(150, 20%, 20%) which translates to approximately #33403A for a muted dark background.
- Accent color: HSL(120, 50%, 60%) which translates to approximately #4CBF50 for interactive elements.
- Font: 'Inter', sans-serif for body and headlines.
- Minimalist outline icons to represent payment options and features.
- Clean, grid-based layout with a focus on readability and user experience.
- Subtle transitions and animations for user interactions, like button presses and link generation.
